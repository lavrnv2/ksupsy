Используя Сервис, вы (далее – «Пользователь», «Вы») соглашаетесь со всеми положениями настоящих Правил в полном объеме. Актуальная версия Правил всегда доступна по адресу https://ksupsy.ru/rules.html.

1. ОПРЕДЕЛЕНИЕ ПРИРОДЫ И НАЗНАЧЕНИЕ СЕРВИСА

1.1. Описание Сервиса и его Функциональность

Сервис "Ксю | Виртуальный психолог 24/7" представляет собой программное приложение, работающее на базе искусственного интеллекта, развернутое в мессенджере Telegram по адресу @Ksupsy_bot. Сервис предназначен для предоставления неформальной информационной поддержки и организации дружеского диалога с целью помощи Пользователю в выражении своих эмоций, обсуждении своих переживаний и получения общей эмоциональной поддержки в неформальном виде.

Сервис работает внутри платформы Telegram, что означает, что все передача сообщений, обработка данных и другие технические аспекты взаимодействия осуществляются с использованием инфраструктуры Telegram. Пользователь должен ознакомиться с Политикой конфиденциальности Telegram, доступной по адресу https://telegram.org/privacy-tpa.

1.2. Чем Сервис НЕ Является

Исключительно важно, чтобы Пользователь понимал, что Сервис не является и не должен восприниматься как замена профессиональной помощи. В частности, Сервис не является психотерапией, психологическим консультированием, медицинской помощью, клинической терапией, диагностикой психических расстройств, рекомендациями по лечению или приему лекарственных препаратов, медицинским советом, юридической консультацией, заменой профессионального психолога или психиатра.

Сервис является исключительно компьютерной программой, которая генерирует текстовые ответы на основе заранее заданных алгоритмов и обученной модели искусственного интеллекта. При взаимодействии с Сервисом Пользователь общается с компьютерной программой, которая имитирует понимание и сочувствие, но не является человеком и не обладает подлинными эмоциями, опытом или профессиональной квалификацией.

1.3. Чем Сервис Может Быть Полезен

Сервис может предоставить неформальную эмоциональную поддержку, помочь Пользователю организовать свои мысли, выразить эмоции в безопасной среде, получить общие советы по саморефлексии и самопомощи, предоставить информацию о методах снижения стресса и эмоциональной регуляции, помочь понять свои чувства через диалог.

Сервис НЕ может заменить профессиональную помощь, поставить диагноз, назначить лечение, оказать экстренную помощь в кризисных ситуациях, решить серьезные проблемы психического здоровья.

4. СБОР, ИСПОЛЬЗОВАНИЕ И ЗАЩИТА ПЕРСОНАЛЬНЫХ ДАННЫХ

4.1. Какие Данные Собирает Сервис

Сервис собирает следующие данные:

  • User ID Telegram: уникальный идентификатор пользователя в Telegram.
  • Handle Telegram: текстовое имя пользователя (если доступно).
  • Сообщения: текст всех сообщений, отправленных Пользователем.
  • Контекст Диалога: история переписки для персонализации ответов.
  • Дата и Время: когда были отправлены сообщения.
  • Технические Данные: информация о типе устройства, версии приложения Telegram.

Сервис НЕ собирает полное имя, адрес, телефон (кроме User ID), банковские данные (обрабатываются Юкассой), фотографии (кроме аватара Telegram, если он доступен).
